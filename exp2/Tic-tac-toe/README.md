
Ex no: 2(c)        Adversarial Search and Game playing
                   TIC-TAC-TOE

Aim: to implement a Tic-Tac-Toe game using the minimax algorithm where AI plays optimally.

Algorithm:

Step 1: initialize an empty 3x3 board.
Step 2: Let user and AI plays alternately.
Step 3: use the minimax function to determine the best move for AI
Step 4: Evaluate the board recursively for ai(Max) and user(Min) moves.
Step 5: check for winning or draw condition after each move
Step 6: Repeat until the game ends.
Step 7: Display the winner or a draw.
