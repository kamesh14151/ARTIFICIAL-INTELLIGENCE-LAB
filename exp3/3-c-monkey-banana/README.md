#  Monkey and Bananas in Prolog

##  Aim
To simulate the **monkey and banana problem** using **logical reasoning** and **action rules**.

---

##  Algorithm

1. **Define state transitions**
   - Actions: `move`, `climb`, `drag`, `walk`

2. **Set goal**
   - Monkey has the banana.

3. **Use recursive rules**
   - Derive the sequence of actions.

4. **Base case**
   - If goal state is reached → return solution.

5. **Recursive case**
   - Apply valid moves to transition states.

6. **Trace the action path**
   - Keep track of the sequence.

7. **Display sequence of actions**
   - Show how the monkey achieves the goal.
