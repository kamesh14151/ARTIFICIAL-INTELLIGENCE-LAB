# ➗ Basic Arithmetic in Prolog

## 🎯 Aim
To implement **basic arithmetic operations** using **first-order logic** in Prolog.

---

## 📚 Algorithm

1. **Define add/3**
   - Performs `X + Y`.

2. **Define sub/3**
   - Performs `X - Y`.

3. **Define multiply/3**
   - Performs `X * Y`.

4. **Define divide/3**
   - Performs `X / Y`, checking `Y ≠ 0`.

5. **Display the result**
   - Output the computed value.
