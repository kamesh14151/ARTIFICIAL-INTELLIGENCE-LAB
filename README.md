# Artificial Intelligence Lab Repository

This repository contains AI lab experiments implemented as part of the Artificial Intelligence and Machine Learning course work. The code is organized experiment-wise, and each experiment folder includes program files, supporting output images (where available), and a dedicated README with experiment-specific explanation.

## Repository Overview

The repository includes implementations in:
- Python
- Java
- Prolog

Main topics covered:
- Search algorithms
- Local optimization
- Game playing strategies
- Logic programming and inference
- Constraint satisfaction
- Information retrieval
- Chatbot systems

## How to Use This Repository

1. Open any experiment folder.
2. Read the local README for that experiment.
3. Run code from the programs directory in that experiment (if present).
4. Refer to output-images for screenshots where provided.
